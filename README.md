# supercommit

supercommit is an utility that helps you quickly commiting your changes, using LLMs for auto-generating meaningful commit messages based on your changes.

## Usage

```bash
pip install .
```

```
supercommit
```

### Args

```bash
supercommit --version
```

```bash
supercommit --path path/to/your/repo
```

## Contributing

```bash
pip install -e .
```

```
supercommit
```
